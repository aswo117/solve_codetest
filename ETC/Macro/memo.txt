==================

레이저 마우스 매크로 (Razer Synapse)를 사용할 때 특정 클라이언트에만 매크로가 동작하게 하는 방법은 다음과 같습니다. Razer Synapse 자체에서는 AHK처럼 창 제목이나 클래스를 기반으로 매크로를 제한하는 직접적인 기능은 제공하지 않지만, 몇 가지 방법을 조합하여 비슷한 효과를 낼 수 있습니다.

1. 프로필 (Profile) 활용:

각 클라이언트별 프로필 생성: Razer Synapse에서 각 프로그램 또는 게임별로 프로필을 생성합니다.

프로필 자동 전환: Razer Synapse에서 특정 실행 파일(exe)을 감지하여 해당 프로필로 자동 전환되도록 설정합니다.

매크로 지정: 각 프로필에 해당 클라이언트에서만 사용할 매크로를 지정합니다.

구체적인 방법:

Razer Synapse 실행: Razer Synapse 프로그램을 실행합니다.

프로필 탭 이동: Synapse UI에서 "프로필" 또는 비슷한 이름의 탭으로 이동합니다.

새 프로필 생성: "새 프로필 추가" 버튼을 클릭하여 새 프로필을 만듭니다.

프로필 이름 설정: 프로필 이름을 해당 클라이언트의 이름으로 설정합니다. (예: "메모장", "게임 A", 등)

프로필 연결: 프로필 설정에서 "연결된 앱" 또는 비슷한 항목을 찾습니다.

실행 파일 지정: 연결된 앱 목록에 해당 클라이언트의 실행 파일(.exe)을 추가합니다.

프로그램 실행 파일은 "찾아보기" 버튼을 통해 직접 선택하거나, 프로그램 실행 중 Synapse UI에서 "추가" 버튼을 통해 자동으로 연결할 수 있습니다.

매크로 설정: 해당 프로필이 활성화되었을 때 사용하고자 하는 매크로를 설정합니다.

매크로 설정은 "마우스", "키보드", 또는 "매크로" 탭에서 설정할 수 있습니다.

다른 클라이언트 프로필 반복: 다른 클라이언트에도 3~7단계를 반복하여 프로필을 생성하고 매크로를 설정합니다.

프로필 자동 전환 확인: 해당 클라이언트를 실행하면 자동으로 해당 프로필로 전환되는지 확인합니다.

장점:

Razer Synapse의 기본 기능을 활용하므로 간단하게 설정 가능합니다.

각 클라이언트별로 다른 매크로 설정이 가능합니다.

다른 소프트웨어 없이 자체적으로 관리 가능합니다.

단점:

실행 파일 기반으로 작동하므로, 창 제목이나 클래스 기반의 정교한 설정은 불가능합니다.

여러 개의 창이 같은 실행 파일을 사용하는 경우, 모든 창에 동일한 매크로가 적용됩니다. (예: 여러 개의 메모장 창)
